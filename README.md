# learn Java

## create develop environment
```
1. 安装jdk
macos: https://res001.geekbang.org/tools/jdk-11.0.2_osx-x64_bin.dmg
windows：https://res001.geekbang.org/tools/jdk-11.0.2_windows-x64_bin.exe

2. 设置Java环境变量
JAVA_HOME: C:\Program Files\Java\jdk-11.0.2
PATH: C:\Program Files\Java\jdk-11.0.2\bin

3. Vscode 安装java插件 Extension Pack for Java
```
